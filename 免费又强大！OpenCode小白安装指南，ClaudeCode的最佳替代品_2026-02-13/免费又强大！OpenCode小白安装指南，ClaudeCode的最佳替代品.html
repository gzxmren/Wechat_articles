一、安装Opencode
OpenCode 是一个开源的AI编程助手，它可以：分析你的代码、帮你写脚本文件、修改和优化项目、支持多种Al模型（Claude、ChatGPT、Gemini、GLM-4.7等）运行在终端中，操作方便简单
简单理解：它就像一个超级聪明的助手，你用自然语言告诉它要做什么，它就会帮你输出文件结果。
Oh My OpenCode 是 OpenCode的增强插件，它为OpenCode添加了更多强大功能：
  • 多AI模型协作：可以同时调用多个AI模型协同工作
  • 智能体系统(Agents)：内置多个专业智能体（如frontend-ui-ux-erngineer、 oracle等）
  • 提示词优化：自动优化你给AI的指令（这点很实用）
  • 后台任务管理：可以并行执行多个任务
简单理解：安装Oh My OpenCode 后，OpenCode 就从一个AI助手变成了一个完整的AI开发团队。
1.1 安装前置工作：
以下全部工具都不要修改安装路径（也就是默认安装在C盘）
安装git
通过命令行工具安装 git 是最简单方便的，这里我们用到 Windows 系统自带的 powershell。
依次输入以下命令（#后面的是注释，不需要输入到命令行中）：
winget --version   #检查 winget 是否可用，如果显示版本号，就可以继续。winget install --id Git.Git -e --source winget   #安装 Git，安装过程会自动弹出安装程序，一路 Next 即可。
安装 Node.js 运行环境

可以“环境”理解成：一个工具仓库

环境A仓库里有锤子、钉子

环境B仓库里有电钻、扳手

你的程序像一个工人：

在A仓库工作 → 就可以使用锤子、钉子，但却用不了B仓库的电钻、扳手

所以只有安装了 Node.js 运行环境，我们才可以通过使用npm 指令，从Node.js 的包管理器中提取Opencode包。
请在浏览器中访问：
https://nodejs.org/zh-cn/download
如果电脑使用的是公司内网，需要联系IT运维开通权限。
1.2 安装 opencode 和安装 Oh My OpenCode
安装opencode
通过命令行工具安装 Opencode 是最简单方便的，这里我们继续使用 Windows 系统自带的 powershell。
npm install -g opencode-ai
  • npm 是 Node.js 的包管理器
  • -g 表示全局安装
  • 安装完成后，可以在任何目录使用 opencode 命令
验证安装
安装完成后，运行以下命令验证是否安装成功：
opencode --version
如果显示版本号（如 1.1.5 或更高），恭喜你，OpenCode 安装成功了！
启动 OpenCode
在 PowerShell 中输入：
opencode
启动后如图所示
按下快捷键 Ctrl + P 打开命令面板，选择 “switch model”，然后可以切换模型
不过实测国产大模型中还是智谱的glm-4.7（最新的是glm-5）推理能力最强，但是要付费。
安装Oh My OpenCode
  1. 打开 PowerShell
  2. 直接运行以下命令：
npx oh-my-opencode install
等待安装完成（可能需要 1-2 分钟）
验证是否安装成功
安装完成后，在 PowerShell 中运行：
opencode
打开opencode后，输入hello，打声招呼，看到ohMyOpenCode的弹窗，说明安装成功
