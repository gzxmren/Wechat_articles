我用AI帮我写了一个专业级A股分析系统


作为一个程序员+业余股民，我一直想要一个属于自己的股票分析工具。市面上的软件要么功能受限，要么收费昂贵，要么数据延迟严重。于是我决定：自己动手，丰衣足食！

📌 为什么要自己做？

先说说我的痛点：

1. 信息不对称

  • 机构有专业的量化团队和数据终端
  • 散户只能看着延迟的行情干瞪眼
  • 想要的功能，免费软件没有；有的软件，动辄几千上万

2. 现有工具的局限

  • 同花顺/东方财富：功能强大但数据要会员
  • 通达信：公式系统强但界面老旧
  • 各种量化平台：学习成本高，不够灵活

3. 程序员的执念

  • 想要什么功能就加什么功能
  • 数据完全自主可控
  • 顺便学习全栈开发

于是，这个项目诞生了 👇

2️⃣ 专业K线图

系统集成了专业的K线图表，支持：

  • 多周期切换：日K/周K/月K/分钟K
  • 技术指标叠加：MACD、KDJ、RSI、布林带
  • 均线系统：MA5/10/20/60/120/250
  • 成交量分析

3️⃣ 资金流向分析（核心功能）

这是我最得意的功能之一：

# 资金流向选股示例
{
    "flow_direction": "inflow",      # 主力流入
    "period_days": 5,                # 5日统计
    "min_flow_amount": 5000,         # 最小流入5000万
    "min_flow_ratio": 1.0,           # 占市值比例>1%
}

能分析什么？

  • 主力资金：大单净流入/流出
  • 多周期对比：1/2/3/5/8/13/21/34天（斐波那契数列）
  • 资金强度评分：综合评估资金流入质量
  • 风险过滤：自动排除ST、停牌、异常股票

4️⃣ 智能选股器

支持多条件组合筛选：

筛选条件示例：
✅ 连续3天阳线
✅ 换手率 > 5%
✅ PE < 30倍
✅ 5日均线上穿10日均线
✅ 主力资金连续3天流入

特色功能：

  • 可视化条件配置（不用写代码）
  • 策略保存与分享
  • 一键重新筛选
  • 结果导出Excel

为什么选择这套技术栈？

技术
选择理由
Vue 3
响应式系统强大，Composition API 写起来爽
FastAPI
Python生态最快的Web框架，自带API文档
PostgreSQL
支持JSONB，适合存储复杂的财务数据
Redis
缓存行情数据，3秒过期，保证实时性
Celery
异步处理数据更新、回测等耗时任务
Tushare
数据全面，API稳定，延迟仅3秒
