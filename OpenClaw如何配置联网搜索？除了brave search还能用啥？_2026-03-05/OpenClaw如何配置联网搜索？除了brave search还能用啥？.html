OpenClaw如何配置联网搜索？除了brave search还能用啥？

web_search，做为openclaw最最最重要的组件之一，配和不配效果简直天差地别

最近这个问题，骁哥每天都要被私聊问50次：“web_search怎么配置”🫨

干脆写直接个攻略，分享给大家🤓

最强搜索工具，brave search

Brave search作为联网搜索首选，是目前最好用的联网搜索引擎，同时也是OpenClaw官方推荐的方案

https://brave.com/zh/search/api/

⚠️不便之处是需要绑visa才能用（国内就行，骁哥用的是国内招行visa）

每个月有5刀，也就是1000次的免费搜索额度，超出额度就需要付费了

骁哥推荐，有visa卡的小伙伴，还是值得去折腾一下的🧏

然后记得去设置一下防护：每月额度最大不超过（比如5刀），防止无限烧钱

申请好API Keys

然后直接配置一下openclaw.json，配置在tools.web.search下

"tools": {
  "web": {
    "search": {
      "provider": "brave",
      "apiKey": "你的apikey"
    }
  },
}

就OK啦

没卡的小伙伴，如果一定一定想要体验，可以去咸鱼搞个token。现在大概是20块钱1000次...不是很值，因为绑卡的话每月1000次是送的...😅

第二强的搜索工具，tavily

https://www.tavily.com/

tavily也是做联网搜索的，效果比起brave略弱一丢丢

但有个好处是，不用绑卡，注册个账号就能用😃

而且同样每个月有免费额度

大家登录后，去https://app.tavily.com/home申请下api key就好了

然后，我们直接跟openclaw说：

“安装下https://github.com/tavily-ai/skills这个skill，安装到.openclaw/skills下”

然后我们可以把api key直接告诉他

不过骁哥更推荐的方式是：直接配置到.env里

在.openclaw目录下，新建一个.env文件

然后直接写进去

TAVILY_API_KEY=你的token

就OK啦！

兜底方案：multi-search-engine

multi search engine是一个开源skill，既不需要api，又不用付费

我们直接跟openclaw说：

“https://github.com/openclaw/skills/blob/main/skills/gpyangyoujun/multi-search-engine/SKILL.md，这个skill也安装下”

就OK啦，就这么简单🤓

最佳实践：三者结合

以上三种联网搜索方案，我们最好结合起来使用

把这段文字，复制到你的Agents.md里

## 联网搜索优先级

1. **Brave Search**（`web_search` 工具）— 首选，速度快、质量高
2. **Tavily Search**（`search` skill）— Brave 额度用完后切换
3. **Multi-Search-Engine Skill**（`web_fetch` 拼 URL）— 最终兜底，无 API 限制

原则：能用 Brave 就不用 Tavily，能用 Tavily 就不用 multi-search-engine

一直白嫖一直爽❗️🤓

当然用不了brave的小伙伴，用tavily + multi-search-engine方案，也是OK的
