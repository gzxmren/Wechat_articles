把精益创业大脑装进终端：Claude Code 极简创业 SKILL 指南

这是一份为 Claude Code 用户准备的极简创业者（The Minimalist Entrepreneur）SKILL 使用指南。SKILL 地址在文末。

关于作者：Sahil Lavingia

Sahil Lavingia 是美国著名的创业家、作者和投资者，以其独到的创业哲学而闻名。他是《小而美——持续盈利的经营法则》（The Minimalist Entrepreneur）一书的作者，该书概括了他多年创业经验的精华。

Sahil Lavingia 最广为人知的成就是创办了 Gumroad —— 一个为创作者提供销售数字产品和内容工具的平台。通过 Gumroad，他积累了丰富的创业管理经验，特别是在如何以最小化的投资和复杂度运营成功企业方面有深刻体悟。

他的创业哲学核心理念是：不需要风险投资、不需要复杂的融资方案，创业者可以通过找到真实的用户社区、验证问题真实性、快速构建最小可行产品、有机地获得第一批客户，从而建立可持续的、盈利的业务。

这套 Skill 将 Sahil Lavingia 在《The Minimalist Entrepreneur》一书中提出的创业框架转化为 Claude Code 可用的交互式工具，帮助创业者按照一套系统、经过验证的方法论来建立和发展他们的业务。

插件概述：Claude Code Skills

这款插件将 Sahil 的创业方法论直接集成到了你的终端（Claude Code）中。它不仅仅是一组命令，更是一个 AI 商业顾问，引导你走过从寻找点子到规模化增长的每一个阶段。

快速安装

请在你的终端中依次执行以下步骤：

  1. 克隆仓库到本地：

    Bash

    git clone https://github.com/slavingia/skills.git ~/.claude/plugins/skills

  2. 在 Claude Code 中加载插件：

    启动 Claude Code 后，输入：

    Bash

    /plugin install ~/.claude/plugins/skills

功能指令指南

该插件涵盖了创业全生命周期的 9 个关键环节。你可以根据自己当前所处的阶段调用相应的指令。

指令适用场景
/find-community寻找社区
：当你还在寻找商业灵感或试图定位目标受众时。
/validate-idea验证想法
：测试你的商业点子是否值得投入精力去实现。
/mvp最小可行产品
：准备构建第一个版本，但不知如何简化功能时。
/first-customers获取首批用户
：已有产品，目标是获取前 100 名种子用户。
/pricing定价策略
：设定价格或考虑调整现有价格模型。
/marketing-plan营销计划
：已实现产品/市场契合（PMF），准备通过内容获客。
/grow-sustainably可持续增长
：在支出、招聘或规模化方面面临决策困境。
/company-values企业价值
：定义文化，为招聘第一批员工做准备。
/minimalist-review极简审查
：对任何商业决策进行“直觉检查”，确保符合极简原则。

创业旅程：如何按步骤使用

极简创业的逻辑是循序渐进的，你可以按照以下路径与 Claude 协作：

1. 从社区出发 (/find-community)

极简创业的第一步不是写代码，而是观察。使用此命令，Claude 会帮你识别你所属的社区，并分析其中尚未被满足的需求。

2. 验证与构建 (/validate-idea & /mvp)

在动手之前，先验证问题是否真实存在。当你准备开发时，/mvp 会无情地砍掉多余功能，强迫你先通过手动流程或最简单的工具来解决问题。

3. 销售与定价 (/first-customers & /pricing)

极简创业者从第一天起就考虑收费。这些指令会帮你制定针对前 100 名用户的“一对一”销售策略，并帮你摆脱“免费陷阱”。

4. 规模化与复盘 (/grow-sustainably & /minimalist-review)

当你开始盈利，压力随之而来。/minimalist-review 是一个非常有用的工具，它会问你：“这真的有必要吗？”、“这会让事情变得更复杂吗？”，帮你保持专注。

小贴士：

在使用这些命令时，请尽量为 Claude 提供更多的背景信息（例如你目前的进展、遇到的困难或具体的行业细节），这样它生成的建议会更具针对性。
