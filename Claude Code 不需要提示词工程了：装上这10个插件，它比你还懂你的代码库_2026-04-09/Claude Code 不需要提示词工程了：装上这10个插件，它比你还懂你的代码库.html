Claude Code 不需要提示词工程了：装上这10个插件，它比你还懂你的代码库

 



如何把 Claude 从"会聊天的"变成"能干活的"

Claude 已经解锁了超能力，但大多数人还在把它当 ChatGPT 用。

自 2026 年 2 月份开始，Anthropic 给 Claude 加了私有插件市场。两周前，社区已经攒了1000 多个 MCP 服务器。现在 Claude 能：

  • • 自己写代码、跑代码
  • • 实时翻网页查资料
  • • 读你的 Gmail、Drive、Slack、数据库
  • • 用 Playwright 真刀真枪操作浏览器
  • • 查任何库的实时文档（不是训练数据里过时的）
  • • 跨对话记住上下文

这不是给 ChatGPT 打补丁。MCP（模型上下文协议）是另一套底层架构——Claude 终于长出了手和脚。

下面这 10 个插件，是我筛掉噪音后真正在用的。

MCP 是什么

Anthropic 2025 年底开源的协议，统一了 AI 连外部工具的方式。以前每个服务都要单独对接，现在一个协议，千种工具，零摩擦。

官方说法：MCP 让 AI 助手能发现和调用本地/远程资源，无需为每个资源定制集成。

人话：装一次，到处用。

10 个真正有用的插件

1. Feature-Dev｜8.9 万+ 安装

干嘛的： 把需求丢给它，7 步走完直接出生产代码

别的 AI 生成代码片段，Feature-Dev 跑的是老工程师的完整流程：

  1. 1. 拆需求
  2. 2. 并行探代码库（多个子智能体同时读你的架构）
  3. 3. 设计架构
  4. 4. 写实现
  5. 5. 补测试
  6. 6. 自审代码
  7. 7. 写文档

用法：敲 /feature-dev，描述你要什么， Claude 自己推进。

适合： 做完整功能，不是拼代码片段

npx skills add anthropic/feature-dev

2. Frontend-Design｜27.7 万+ 安装

干嘛的： 让 Claude 写 CSS 之前先长设计脑子

没它，Claude 生成的界面一眼"AI 味"——平庸、雷同、安全牌。

装了之后它会考虑：

  • • 这东西是干嘛的，该有什么调性
  • • 故意加约束（比如"只用两种字体"）
  • • 视觉差异化
  • • 出挑的字体搭配
  • • 非对称布局
  • • 滚动触发的微动效

有开发者说："别人问我设计师是谁。没有设计师，就是 Claude 装了 frontend-design。"

适合： 受够了 AI 审美，想要"像人设计的"

npx skills add anthropic/frontend-design

3. Context7（Upstash）

干嘛的： 让 Claude 读实时、指定版本的库文档

Claude 的训练数据有截止日期。你用 Next.js 15、React 19、Tailwind 4，它可能按旧版 API 给你幻觉代码。

Context7 查的是你当前装的确切版本的文档。

两个核心工具：

  • • resolve-library-id —— 找库
  • • resolve-library-id —— 捞文档

现在 Claude 会说"根据 Next.js 15 文档…"，而不是凭记忆猜。

适合： 追新版本的框架党

安装：Claude Desktop 配置里加 MCP

4. GitHub MCP Server

干嘛的： 搜仓库、读代码、开 issue、提 PR

不用手动拷代码片段，直接说：

"分析 src/ 目录，给 #42 提个修复方案"

Claude 会自己搜仓库、读文件、给方案、开 PR。

适合： 用 GitHub 的开发者

安装：Anthropic 官方市场

5. PostgreSQL MCP

干嘛的： 只读访问你的 Postgres 数据库

自然语言提问：

"上个月消费最高的 5 个用户是谁"

Claude 自动扫 schema、生成 SQL、执行、返结果。不用背表名，不用调语法。

适合： 分析师、工程师

⚠️ 只读权限，防手滑

6. Playwright MCP

干嘛的： Claude 控制真 Chrome 窗口测你的 Web 应用

不用写测试脚本，口述：

"测下单流程：浏览商品、加购物车、填测试卡、支付、确认成功"

Claude 打开可见的浏览器窗口，你看着它点。

隐藏技巧： 需要登录的功能，你先手动登进去，Claude 接着跑。

适合： 前端工程师做 E2E 测试

⚠️ 吃资源，别在老爷机上开

7. Brave Search MCP

干嘛的： 给 Claude 接实时互联网

Claude 的知识截止到 2025 年 1 月。Brave Search 让它能：

  • • 查新闻
  • • 看股价
  • • 找最新库文档
  • • 实时核实

"这周 AI 有什么新发布？"

Claude 现搜现答。

适合： 研究、事实核查、追热点

8. Google Workspace（gws CLI）｜3 天 4900 Star

干嘛的： 一条命令打通 Drive、Gmail、Calendar、Sheets

以前接 Google 服务，每个都要单独配 OAuth。Google 出了 gws，全打包了。

npm install -g @googleworkspace/cli
gws mcp -s drive,gmail,calendar,sheets

现在 Claude 能直接读你的整个 Workspace。

示例工作流：

"读未读邮件，紧急的草拟回复，更新季度指标表，给临近截止日的任务加日历提醒，生成摘要文档"

一个提示，零复制粘贴。

适合： 重度 Google 用户

9. Slack MCP Server

干嘛的： 读频道历史、总结线程、发消息

2026 年 1 月更新加了交互式起草：

  1. 1. 在 Claude 里写草稿
  2. 2. 预览
  3. 3. 改完再发

冷门用法：

"总结 #engineering 频道这两天聊了什么，我再回复"

不用翻 500 条消息，上下文直接到手。

适合： 跨多个 Slack 工作区的人

10. Memory Bank / Claude-Mem

干嘛的： 让 Claude跨对话记住东西

LLM 本来聊完就忘。Memory Bank 建了个 .memory 文件夹，存：

  • • 你的偏好
  • • 项目细节
  • • 过往决策
  • • 代码规范

结果：Claude 永久记得你的认证模式、错误处理约定、测试要求。

解释一次，终身受用。

适合： 长期项目，上下文值钱

三种安装方式

方法一：Claude Desktop 配置（MCP 服务器）

改 claude_desktop_config.json：

{
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres", "postgresql://localhost/mydb"]
    }
  }
}

重启 Claude Desktop。

方法二：npm 装 Skills

npx skills add anthropic/frontend-design
npx skills add anthropic/feature-dev

方法三：Claude Code 插件系统

/plugin install feature-dev@claude-plugins-official
