Antigravity 必装的 4 个神级插件，用过都说好
前几天写了一篇关于《Antigravity》的文章，有好多小伙伴问有没有合适推荐的插件，这不就来了吗
如果你是 Antigravity 的重度用户，尤其是会同时跑多个 Agent、频繁切换模型的那种，这篇文章专门为你准备。下面这 4 个插件，能实实在在地改善你的日常工作流，建议收藏备用。

01 中文语言包——让界面说人话

插件简介

Antigravity 默认界面为英文，对于非英语母语用户来说，安装中文插件是第一步，也是最重要的一步。安装后，所有菜单、设置项、提示文字都会切换为简体中文，大幅降低上手门槛。

如何安装

打开 Antigravity，进入顶部菜单栏，点击「扩展」→「插件市场」
在搜索框中输入「中文（简体）」
找到对应插件后，点击「安装」按钮
安装完成后，按照提示重启 Antigravity，界面即切换为中文

使用建议

强烈建议在安装任何其他插件之前，先装好中文语言包。这样后续所有插件的设置界面也能直接以中文显示，省去逐项翻译的麻烦。

02 Antigravity Cockpit——配额监控仪表盘

插件简介

这款插件能实时监控你在 Antigravity 中各模型的使用配额，是多模型并发用户的必备工具。尤其当你同时使用多个 Agent、跨模型调度任务时，它能帮你提前规划用量，避免在关键节点踩到配额上限。

如何安装

打开「插件市场」
搜索「Antigravity Cockpit」
点击「安装」，等待安装完成后重启

如何使用

安装完成后，注意观察 Antigravity 右下角的状态栏，会出现新的图标
点击该图标，选择「打开配额监控面板」
面板中会清晰列出每个模型的剩余配额及所属分组（Group）

重要说明

Antigravity 对模型采用分组（Group）计费机制，同一 Group 内的模型共享同一个配额池。举个例子：如果 Claude Opus 4.6 和 Claude Sonnet 4.6 同属一个 Group，那么 Opus 用满额后，Sonnet 也会随之不可用——这是很多用户感到困惑的原因所在。Cockpit 面板会直观地展示这种分组关系，让你对用量一目了然。
另外，请注意：目前最新版 Antigravity 中，Gemini 3.1 已上线，Gemini 3.0 Pro 已下线，请及时更新使用习惯。

03 Antigravity Quota（AGQ）——轻量级配额显示

插件简介

AGQ 的功能与 Cockpit 类似，同样用于实时显示 AI 模型的使用量，但更加轻量，不会占用额外的面板空间，直接在状态栏以数字形式呈现配额信息。

如何安装

打开「插件市场」
搜索「Antigravity Quota」或「AGQ」
点击「安装」，重启后生效

如何使用

安装后无需额外配置，状态栏会自动出现实时配额数字。将鼠标悬停在数字上，可查看简要的配额详情。

选哪个好？

如果你希望有完整的可视化面板和分组信息，选Cockpit；如果你只需要一个简洁的状态栏数字提示，选AGQ即可，两者按需二选一。

04 Graph-It-Live——代码依赖关系实时可视化

插件简介

这是四款插件中技术含量最高的一个。它的核心功能是：当多个 Agent 并发修改代码时，实时将文件之间的依赖关系渲染成可交互的图表。
在单 Agent 工作流中，它的价值可能有限。但一旦你开始让多个 Agent 同时操作不同模块，代码结构的变化就会变得难以追踪——某个 Agent 修改了一个底层文件，你很难立刻意识到这会影响哪些上层依赖。Graph-It-Live 把这个动态过程可视化，让你在问题扩散之前就能发现结构性风险。

如何安装

打开「插件市场」
在搜索框中输入该插件的作者名字进行搜索（注意：直接搜插件名可能找不到，建议搜作者）
找到 Graph-It-Live 后点击「安装」，重启 Antigravity

如何使用

安装完成后，按照以下步骤启用并打开插件：
第一步：启用 MCP Server
按下Ctrl + Shift + P唤出命令面板，在输入框中键入：
Open Settings
打开设置后，搜索并找到以下配置项：
graph-it-live.enableMcpServer
将其值设置为enable（启用）。
第二步：打开项目并调出图表
打开一个结构较为复杂的项目，再次按下Ctrl + Shift + P唤出命令面板，在输入框中键入Graph或Graph-it，即可找到并启动 Graph-It-Live 插件。
第三步：查看依赖图
插件会自动分析当前项目的文件依赖关系，并渲染成可交互的动态图表。你可以在图中点击节点查看具体文件，也可以实时观察 Agent 修改代码时依赖关系的变化。请耐心等待一段时间。

写在最后

这 4 个插件覆盖了 Antigravity 使用中最常见的几类需求：界面汉化、配额监控、轻量提示和结构可视化。根据自己的使用习惯按需安装即可，不必全部装上。如果你也有其他好用的插件推荐，欢迎在评论区留言交流。
