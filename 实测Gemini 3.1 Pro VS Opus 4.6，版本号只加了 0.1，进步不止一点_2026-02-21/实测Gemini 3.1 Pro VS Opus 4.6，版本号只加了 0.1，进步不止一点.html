总结
01

交互解谜网页/ 看视频理解需求

需求刷小红书看到一个很酷的书籍解谜交互网页（百年孤独），想让 AI 照着视频复刻一个，换成加缪的《局外人》。

Prompt

参考这个小红书视频做一个《局外人》交互解密书籍网页

这个场景考验的是"看视频理解需求"的能力。整体来看，3.1 Pro 的交互更有趣，内容也更完整，对视频里的书籍解谜形式还原得最好。Opus 跑偏了，没按视频里的书籍形式来，自己发挥了一套。这里要提一下，Gemini 是原生支持视频输入的，能直接"看"视频理解内容；而 Claude 系列目前还只能把视频拆成一帧帧图片去分析，理解深度有差距。

02

看视频复刻手机 UI/ 视频理解能力

需求给一段手机 App 的展示视频，让 AI 直接复刻出 UI 界面和交互。

上一场我们发现 Opus 没法直接看视频，所以这次专门给它加了一句提示词，让它把视频拆解为一帧帧图片再查看内容。给它一个公平竞争的机会。

Prompt

帮我复刻这个 UI 设计稿以及交互

原视频：

给了 Opus 拆帧提示之后，它反而做得最全面，内容完整度最高。但因为是拆成一帧帧图片看的，视频里的一些交互方式它就学不到了。3.1 Pro 能原生看视频，所以交互细节更像原版，还原度更高。3 Pro 整体也还行，但有一些小问题，可能是 3.1 刚出还不够稳定，3 Pro 反而表现得比较均衡

03

SVG 动画/ 从静态到交互

SVG 一直是 Gemini 的传统强项，从静态绘画到复杂交互动画，难度递进

Case 1日出月落城市动画

让 AI 画一个城市场景的日夜循环动画——太阳下沉、月亮升起、天空渐变、窗灯点亮。

Prompt

请生成单文件 SVG 场景动画：太阳下沉、月亮升起、天空渐变变化、城市窗灯逐步点亮，再反向回到白天并无缝循环。要求颜色过渡平滑，层次明确，代码结构清晰可读

这个场景三个模型差距不大。3.1 Pro 和 Opus 都能交出不错的答卷，各有特色，3 Pro 的太阳和月亮运动很奇怪

Case 2旗帜飘扬交互动画

做一个带交互的旗帜飘扬动画，默认有风摆效果，鼠标靠近时波动加强。

Prompt

请实现单文件 SVG 旗帜动画：默认有持续风摆，鼠标/触摸靠近时局部波动幅度增强并逐渐衰减。要求交互不打断主时间线，移动端触摸可用，离开后恢复常态并继续无缝循环

3.1 Pro 的旗帜更有立体感，波浪纹理和飘动细节都更真实。3 Pro 也不差了。Opus 这场差了不少，旗帜比较扁平。

04

复刻网页/ 还原度测试

需求给一个日本糖果品牌的官网地址，让 AI 直接复刻出来

Prompt

复刻这个网页 https://www.nobel.co.jp/

这场 3.1 Pro 直接拿了满分。它学聪明了，不是从零开始写，而是直接下载原网页进行本地化，还原度极高。Opus 也很完整，3 Pro 则复刻了大致的布局和风格。这个思路本身就说明 3.1 Pro 的"解题策略"变强了，不只是能力提升，思路也更聪明。

05

营销落地页/ 综合前端实力

需求构建一个高端 Vite + React 营销落地页，多模态 AI 提示词工程工作台风格。

Prompt

为 "DreamWeaver" 构建一个高端 Vite + React 营销落地页，定位是多模态 AI 提示词工程工作台

复杂落地页这种场景，三个顶级模型差距已经很小了，都能交出一个像样的作品。我个人更喜欢 3.1 Pro 的展示效果，更清晰一些，但说实话差距不大，细节上各有偏好。

06

网页台球游戏/ 逻辑密集型

需求让 AI 直接做一个网页版的台球游戏，能玩的那种

Prompt

制作一个网页版的专业台球游戏

终于到了 Opus 反超的场景。Opus 不仅做出了可以玩的台球游戏，还多做了比分系统和力度条，完成度最高。3.1 Pro 也能玩了，有球杆有瞄准线，但功能没 Opus 全。3 Pro 就只做了个物理演示，没有完整的游戏逻辑。台球游戏涉及物理引擎、碰撞检测、游戏状态管理，这些都是逻辑密集型的任务。在这种需要"理性思考"的场景里，Opus 4.6 的优势就体现出来了

SUMMARY/ 写在最后

6 个场景测下来，结论挺清晰的：

SVG 和前端，Gemini 3.1 Pro 依然一骑绝尘。尤其是 SVG 绘画和动画，进步非常明显。复刻网页很强，常规操作

但逻辑复杂的任务，Opus 4.6 还是更稳。台球游戏这种涉及物理引擎和游戏状态的场景，Opus 明显想得更周全。包括对长提示词逻辑遵循程度，Opus 的前端能力其实也不弱，只是在 SVG 这种"视觉审美"的活上差了 Gemini 一截。

值得一提的是 Gemini 的多模态优势。第一个场景里，3.1 Pro 能直接看懂小红书视频然后复刻网页，这不是靠"猜"的，而是原生理解视频内容。在顶级模型的竞争里，谷歌走的是全模态路线——能读图、能直接看视频（最长 45 分钟）、能听音频（最长 8.4 小时），不像 Claude 只能把视频拆成一帧帧截图去分析。这是目前谷歌模型最独特的优势

按这个节奏，后面可能还会有 3.2、3.3，每次不一定是颠覆性变化，但每次都在变好

想体验的话，主要有这几个入口：

-Gemini App：最方便，免费用户直接用，日常聊天、问问题够了

-AI Studio：想认真做点东西的话，建议来这里。满血版 3.1 Pro 免费可用

-NotebookLM：做知识管理、读书笔记的，3.1 Pro 加持后知识库更聪明了
