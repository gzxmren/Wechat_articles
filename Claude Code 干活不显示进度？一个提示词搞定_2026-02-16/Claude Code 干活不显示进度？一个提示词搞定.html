Claude Code 干活不显示进度？一个提示词搞定

Claude Code 干活儿老是不显示进度，等着着急。

在 CLAUDE.md 里面加了一个工作模式：执行任务的时候在指令前面添加「/进度」，AI 就会把任务拆成小步骤，每一步都详细汇报。

使用示例

/进度 整理所有文件 /进度 批量处理 100 个图片 /进度 分析所有数据文件

原理

Claude Code 本身就是流式输出，这个提示词只是让 AI 更频繁地输出中间状态。

利用现有特性，不需要额外工具。
